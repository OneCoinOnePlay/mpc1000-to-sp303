# Contributing to MPC1000 to SP-303 Converter

Thank you for your interest in contributing! This project was created for the MPC/SP-303 community.

## How to Contribute

### Reporting Issues

If you encounter a bug or have a feature request:

1. **Check existing issues** first to avoid duplicates
2. **Provide details:**
   - Python version
   - Operating system
   - MPC model (if relevant)
   - SP-303 model/variant
   - Error messages or unexpected behavior
   - Sample .pgm file (if possible)

### Suggesting Enhancements

We welcome suggestions for:
- Support for other MPC models (MPC500, MPC2500, etc.)
- Support for other samplers (SP-404, SP-555, etc.)
- Performance improvements
- Better error handling
- Documentation improvements

### Code Contributions

1. **Fork the repository**
2. **Create a feature branch:** `git checkout -b feature/your-feature-name`
3. **Make your changes:**
   - Keep code simple and readable
   - Add comments for complex logic
   - Follow existing code style
4. **Test your changes** on real hardware if possible
5. **Commit with clear messages:** `git commit -m "Add feature: description"`
6. **Push to your fork:** `git push origin feature/your-feature-name`
7. **Open a Pull Request**

### Code Style

- Use descriptive variable names
- Add docstrings to functions
- Keep functions focused and single-purpose
- Maintain Python 3.6+ compatibility
- Avoid external dependencies when possible

### Testing

If you're testing changes:
- Test with various .pgm files if possible
- Test with different sample durations
- Test on actual SP-303 hardware (ideal)
- Document your test results

### Documentation

Help improve documentation:
- Fix typos or unclear instructions
- Add examples
- Improve troubleshooting section
- Translate to other languages

## Community Guidelines

- Be respectful and constructive
- Help others learn
- Share your results and discoveries
- Credit others' contributions

## Questions?

Open an issue labeled "question" - we're here to help!

## Hardware Testing

If you have access to different SP-303 variants or other samplers and can test compatibility, that's incredibly valuable! Please share your findings.
